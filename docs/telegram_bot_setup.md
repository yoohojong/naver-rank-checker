# 📱 텔레그램 보고 받기 — 사장님 1회 설정 (약 5분)

상노체크 결과를 사장님 텔레그램으로 받기 위한 **딱 한 번만** 하는 설정입니다.
(키워드·순위가 들어가므로 **사장님만 보는 개인 텔레그램**으로 옵니다.)

---

## 1단계 — 봇 만들기 (토큰 받기)

1. 텔레그램 앱에서 검색창에 **`@BotFather`** 입력 → 파란 체크 달린 공식 봇 선택
2. **`/newbot`** 입력
3. 봇 이름 입력 (아무거나, 예: `상노 알리미`)
4. 봇 아이디 입력 — 반드시 **`bot`으로 끝나야 함** (예: `sangno_alrim_bot`)
5. BotFather가 주는 메시지에서 **토큰**을 복사
   - 토큰 모양: `123456789:AAExxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx` (예시 — 실제값 다름)
   - ⚠️ 이 토큰은 **비밀번호**입니다. 아무한테도 보내지 마세요.

## 2단계 — 내 채팅 아이디(chat_id) 알아내기

1. 방금 만든 내 봇을 텔레그램에서 열고 **`/start`** 또는 아무 말이나 한 번 보냄
2. 검색창에 **`@userinfobot`** 입력 → 시작 → 내 **숫자 아이디**(`Id: 123456789`)를 알려줌
   - 이 숫자가 **chat_id**입니다.

## 3단계 — GitHub에 등록 (값 2개)

저장소 → **Settings → Secrets and variables → Actions → New repository secret**에서 2개 등록:

| 이름 (Name) | 값 (Secret) |
|---|---|
| `TELEGRAM_BOT_TOKEN` | 1단계에서 받은 토큰 |
| `TELEGRAM_CHAT_ID` | 2단계에서 받은 숫자 아이디 |

> 두 값을 등록하면 다음 실행부터 텔레그램으로 보고가 옵니다.
> **등록 안 해도 프로그램은 그대로 작동**합니다 (텔레그램만 조용히 건너뜀).
>
> 📌 **저녁/아침 요약** 보고는 GitHub 백업을 읽어오므로 `ACTIONS_PAT` 시크릿도 필요합니다 —
> 기존 자동체크(rank-check)에서 이미 쓰고 있으면 추가 등록 불필요합니다.

---

## 확인 방법
- 저장소 → Actions → 워크플로우 수동 실행(Run workflow) → 잠시 뒤 텔레그램에 메시지가 오면 성공.
- 안 오면: ① 두 시크릿 이름 철자 확인 ② 봇에게 `/start` 한 번 눌렀는지 확인.

## 막히면
- 토큰을 다시 받고 싶으면 BotFather에서 **`/token`** → 봇 선택.
- chat_id가 헷갈리면 `@userinfobot` 다시 실행.
