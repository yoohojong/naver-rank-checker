# 정확도 검증 가이드 (사장님 직접 수행)

## 언제 사용
- cron 직후 (1시간 이내)
- 분기 1회 (T-M42)

## 절차 (15분)
1. GitHub Actions cron 실행 완료 확인 (issue #1 댓글 또는 시트 P1 셀 'cron 갱신' 확인)
2. 시트의 임의 10 키워드 선택 (다양한 카외 탭 + 다양한 K 값)
3. 네이버 PC 검색 (각 키워드 검색 후 본인 카페 글 노출 + 순위 확인)
4. 시트 K/L/M/J 컬럼 vs 본인 수기 비교
5. 불일치 행 = 캡처 + Claude 보고

## 의무 사실
- 시점 차이 = 1시간 이내 검증 의무 (네이버 결과 변동)
- 1페이지만 검증 (parser = 1페이지만 분석)
- 본인 카페 글 (CAFE_WHITELIST 26 slug) 만 검증
