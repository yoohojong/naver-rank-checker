# 경쟁사 리스트업 — 누가 우리 자리를 차지하고 있나

**요청(2026-07-23)**: "누락시킨거 보고 상위노출된 경쟁사 리스트업 (매번 갱신) / 제일 많이 보이는 애들 횟수 체크해서 갱신해주는 시스템"

## 무엇을 해주나

순위 검사기가 키워드를 검색할 때, 화면 위쪽 자리(AB·스마트블록·인기글)에 실제로 올라와 있는 글 목록을
같이 기록합니다. 우리 글은 빼고 남는 것이 **경쟁사**입니다. 매 실행마다 갱신됩니다.

- 수집 범위: **전 키워드** (우리가 누락된 키워드뿐 아니라 1등인 키워드까지)
- 우리 글 제외 기준: 시트에 있는 우리 링크 + 우리 카페 목록(CAFE_WHITELIST)
- 한 곳이 같은 키워드에 여러 글을 깔아도 **1회**로 셉니다(가장 높은 자리 기준)

## 어디서 보나

비공개 시트에 탭 2개가 생깁니다. 사장님 작업 탭은 건드리지 않습니다.

**경쟁사_랭킹** — 사장님이 볼 화면. 매 실행 전체 갱신.

| 주체 | 종류 | 등장 횟수 | 노출 키워드 수 | 평균 순위 | 1위 횟수 | 우리가 놓친 키워드 수 | 최근 등장일 | 대표 URL |
|---|---|---|---|---|---|---|---|---|

- **등장 횟수** = (날짜 × 키워드) 조합. 같은 날 같은 키워드는 1회.
- **우리가 놓친 키워드 수** = 그 경쟁사가 보인 키워드 중, 우리 상태가 누락·미노출·삭제인 키워드 수.
  이 숫자가 큰 곳이 **우리 자리를 실제로 먹고 있는 곳**입니다.

**경쟁사_이력** — 원본 기록(날짜·키워드·구좌·순위·주체·제목·URL). 최근 30일치 보관, 오래된 날짜는 자동 정리.
"누락 건만" 보고 싶으면 이 탭에서 `우리상태 = 누락` 으로 필터하면 됩니다.

- 검색 요청 추가 **없음**. 순위 검사할 때 이미 받아온 화면을 그대로 다시 읽습니다.
- 실행 시간 영향 거의 없음. 늘어나는 건 시트 쓰기뿐(하루 1회 적재 + 랭킹 갱신).
- 수집·기록이 실패해도 순위 검사와 시트 갱신은 그대로 진행됩니다.
