# 상위노출 시트 — 키워드 색상 표기 안내 (사장님·직원용)

## 한 줄 요약
**카페 구좌 1등을 못 하는 키워드**에 색이 칠해지고, **여러 번 발행해도 계속 1등을 못 할수록 색이 진해집니다.** 색이 진한 키워드 = 경쟁이 세거나 더 강한 글(지수)이 필요한 키워드입니다.

## 왜 보는가
카페 구좌 1등을 못 한다는 건 **경쟁사가 그 자리를 차지**하고 있다는 뜻입니다. 즉 그 키워드는 경쟁이 치열하거나, 더 지수 높은 글이 있어야 이길 수 있습니다. 색이 진한 키워드부터 **전략을 다시 볼 대상**(더 강한 글·다른 각도)입니다.

## 색이 뜻하는 것 (키워드 셀 배경색)

| 색 | 뜻 |
|----|----|
| **없음(흰색)** | 아직 판단 전(**발행 당일·다음날**은 지켜보는 중) / 작업일 미입력 / 또는 실패 전적 없이 1등 |
| 🟡 **연노랑** | **1회** — 발행 후 하루가 온전히 지났는데도(발행 다음다음날) 1등 실패 |
| 🟠 **주황** | **2회** — 재발행했는데 또 1등 실패 |
| 🔴 **빨강** | **3회 이상** — 여러 번 재발행해도 계속 1등 실패 = 경쟁 매우 치열 |
| ⬛ **연회색** | 지금은 1등이지만 **과거에 실패 전적**이 있던 키워드(유지 관리 대상) |

## '1일'은 무조건 발행일 기준 (중요)
- 여기서 말하는 경과일은 **오직 작업일(발행일) 기준**입니다. 검사한 날짜나 다른 날짜가 아닙니다.
- **발행 당일과 그 다음날까지는 무조건 흰색**(아직 판단 전).
- **발행 다음다음날(하루가 온전히 지난 시점)부터** 1등 실패면 색이 붙습니다.
  - 예: **7/15 발행 → 7/16 흰색 → 7/17부터** 1등 실패면 🟡 연노랑.

## 횟수는 어떻게 세나
- 키워드를 **발행/재발행(작업일이 바뀜)** 한 뒤 **발행 다음다음날에도 카페 1등이 아니면 +1**.
- 같은 발행분(작업일이 그대로)은 검사가 여러 번 돌아도 **1회만** 셉니다(중복 안 함).
- **카페 1등을 달성하면 0으로 리셋**(진한 색 사라짐) — 해결된 걸로 봅니다. 단, 과거 실패 전적은 ⬛ 연회색으로 남습니다.

## 전적(연회색)은 언제 사라지나
- 실패 전적이 있어 ⬛ 연회색인 키워드도, **14일 이상 계속 1등을 유지하면 전적을 지우고 흰색**으로 돌아갑니다(완전히 안정된 걸로 봄).

## 직원 활용법
- 🔴 빨강 / 🟠 주황 키워드 = 경쟁사가 1등 점유 중. **더 강한 글·다른 접근**이 필요. 우선 검토 대상.
- 🟡 연노랑 = 한 번 실패, 지켜보는 중.
- ⬛ 연회색 = 예전엔 어려웠지만 지금은 1등 — 유지 관리 대상.
- 흰색 = 1등 성공했거나 아직 판단 전 — 정상.

*판정 시점(발행 다음다음날), 색 기준(1/2/3회), 전적 만료(14일)는 원하시면 조정 가능합니다. 문의는 사장님께.*
