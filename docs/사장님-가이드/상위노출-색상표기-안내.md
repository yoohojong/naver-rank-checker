# 상위노출 시트 — 키워드 색상 표기 안내 (사장님·직원용)

## 한 줄 요약
**카페 구좌 1위를 "한 번도 못해본" 키워드**에만 색이 칠해지고, **여러 번 발행해도 계속 1위를 못 할수록 색이 진해집니다.** 색이 진한 키워드 = 경쟁이 세거나 더 강한 글(지수)이 필요한 키워드입니다.

## 핵심 원칙 (가장 중요)
- **색칠(🟡🟠🔴)된 키워드 = 카페 구좌 1위를 한 번도 달성 못해본 키워드.**
- **한 번이라도 카페 구좌 1위에 오른 적이 있으면, 그 뒤에 얼마나 빨리 떨어지든(누락되든) 다시 색칠하지 않습니다.** 한 번 정복해봤으면 "할 수 있는 키워드"로 보고 신경 끄는 겁니다.
- 그래서 빨강·주황·노랑에 있는 키워드는 **"아직 한 번도 1위를 못 잡은, 진짜 어려운 키워드"**만 남습니다.

## 왜 보는가
카페 구좌 1위를 한 번도 못 잡았다는 건 **그 자리를 계속 경쟁사가 차지**하고 있다는 뜻입니다. 색이 진할수록 여러 번 도전(재발행)했는데도 못 이긴 것 → **더 강한 글·다른 각도로 우선 재공략**할 대상입니다.

## 색이 뜻하는 것 (키워드 셀 배경색)

| 색 | 뜻 |
|----|----|
| **없음(흰색)** | 아직 판단 전(**발행 당일·다음날**은 지켜보는 중) / 작업일 미입력 / 또는 실패 이력 없이 1위 달성 |
| 🟡 **연노랑** | **1회** — 1위 무경험 + 발행 후 하루가 온전히 지났는데도(발행 다음다음날) 1위 실패 |
| 🟠 **주황** | **2회** — 재발행했는데 또 1위 실패 (여전히 1위 무경험) |
| 🔴 **빨강** | **3회 이상** — 여러 번 재발행해도 계속 1위 실패 = 경쟁 매우 치열 |
| ⬛ **연회색** | **과거에 1위를 달성한 적이 있는** 키워드(지금 떨어져 있어도). "한 번은 정복함" 표시 |

## '1일'은 무조건 발행일 기준
- 경과일은 **오직 작업일(발행일) 기준**입니다. 검사한 날짜나 다른 날짜가 아닙니다.
- **발행 당일과 그 다음날까지는 무조건 흰색**(아직 판단 전).
- **발행 다음다음날(하루가 온전히 지난 시점)부터** 1위 실패면 색이 붙습니다.
  - 예: **7/15 발행 → 7/16 흰색 → 7/17부터** (1위 무경험이라면) 🟡 연노랑.

## 횟수는 어떻게 세나 (1위 무경험 키워드만 해당)
- 키워드를 **발행/재발행(작업일이 바뀜)** 한 뒤 **발행 다음다음날에도 1위가 아니면 +1**.
- 같은 발행분(작업일 그대로)은 검사가 여러 번 돌아도 **1회만** 셉니다.
- **한 번이라도 1위를 달성하면 그 뒤로는 카운트가 멈춥니다**(떨어져도 다시 안 셈) → ⬛ 연회색으로 넘어갑니다.

## 전적(연회색)은 언제 흰색으로 사라지나
- 과거 실패 이력이 있어 ⬛ 연회색인 키워드도, **3일 이상 계속 1위를 유지하면 전적을 지우고 흰색**으로 돌아갑니다(안정된 걸로 봄). (2026-07-18: 14일→3일 단축)

## 직원 활용법
- 🔴 빨강 / 🟠 주황 = **한 번도 1위를 못 잡은** 어려운 키워드. **더 강한 글·다른 접근** 필요. 우선 검토.
- 🟡 연노랑 = 1위 무경험, 한 번 실패 — 지켜보는 중.
- ⬛ 연회색 = 예전에 1위 해본 키워드 — 신경 안 써도 됨(유지 관리만).
- 흰색 = 안정적으로 1위거나 아직 판단 전 — 정상.
