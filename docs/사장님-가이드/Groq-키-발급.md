# Groq 무료 키 발급 가이드 (텔레그램 봇 "자연어 이해" 켜기)

> 이걸 하면 봇이 **정해진 단어 말고 아무렇게나 물어봐도** 알아듣습니다.
> 예) "샴푸 요새 어때?", "빠진 거 있나?", "비듬샴푸 지금 잘 잡혀?"
>
> **완전 무료입니다. 신용카드 안 넣습니다.** 사장님은 키 1개만 만들어 주시면 되고, 약 **3분**이면 끝나요.

---

## 1단계 — Groq 가입 (1분)

1. 인터넷에서 **https://console.groq.com** 접속
2. **Sign in / Log in** 클릭 → **구글 계정으로 로그인**(또는 이메일)
   - ※ 결제·카드 정보 묻는 화면 **없습니다**. 나오면 멈추고 저한테 알려주세요.

## 2단계 — 키 만들기 (1분)

1. 로그인 후 왼쪽(또는 위) 메뉴에서 **API Keys** 클릭
   - 바로 가기: **https://console.groq.com/keys**
2. **Create API Key**(키 만들기) 버튼 클릭
3. 이름은 아무거나(예: `sangno-bot`) → 만들기
4. 화면에 **긴 글자(키)** 가 뜹니다. `gsk_` 로 시작해요.
   **이 화면에서만 보입니다 → 꼭 복사**해 두세요. (메모장에 잠깐 붙여두기)

## 3단계 — 키를 봇에 등록 (1분)

키를 안전한 곳(GitHub 비밀저장소)에 넣어야 봇이 씁니다. 두 방법 중 편한 걸로:

**방법 A — 사장님이 직접 (웹에서 클릭만)**
1. **https://github.com/yoohojong/naver-rank-checker/settings/secrets/actions** 접속
2. **New repository secret** 클릭
3. Name(이름)에 정확히: `GROQ_API_KEY`
4. Secret(값)에 2단계에서 복사한 `gsk_...` 키 붙여넣기 → **Add secret**

**방법 B — 저한테 키만 주기**
- 채팅으로 `gsk_...` 키를 주시면 제가 등록해 드립니다.
  (단, 채팅에 키가 남으니 신경 쓰이시면 방법 A 추천)

---

## 끝! 그다음은 자동

- 등록되면 다음 봇 점검(5분 주기)부터 **자연어 이해가 켜집니다.**
- 테스트: 봇에게 `샴푸 요새 어때?` 처럼 자유롭게 보내보세요.

## 자주 묻는 것

- **돈 나가요?** 안 나갑니다. 카드가 없어서 구조적으로 과금이 안 돼요. 너무 많이 쓰면 잠깐 멈췄다 풀릴 뿐(우리는 하루 수십 번이라 한참 남음).
- **내 데이터가 새나요?** 봇은 Groq에 **사장님 질문 글만** 보냅니다. 실제 순위·시트 내용·제품 목록은 **안 보냅니다**(답은 봇 안에서 만듦).
- **키를 안 넣으면?** 봇은 그대로 잘 돕니다 — 다만 예전처럼 **정해진 단어**(누락/삭제/순위/요약 등)로만 알아들어요.
- **키가 막히면(만료 등)?** 봇이 자동으로 예전 방식(정해진 단어)으로 되돌아가서, 절대 먹통이 되지 않습니다.
